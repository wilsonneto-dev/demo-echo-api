## Demo Echo API

Docker image: `wilsonnetodev/demo-echo-api`

Just a simple echo API

Hit the `GET /` and see the belwo response:

```json
{
  "version": "v1",
  "hostName": "Lab",
  "assemblyVersion": "1.0.0.0",
  "counter": 1
}
```
